

# Chrome Extension Development - A Full-Stack Approach

## by Kyle Goslin



Welcome to the official code repository for the book **Chrome Extension Development - A Full-Stack Approach**.

Here you can find the source code for each example, ordered by chapter.

In most cases, inside the example folders, the code is broken into two folders, one for the extension and another for the associated API server code.



For additional documentation, please visit the book’s website at [https://www.extensionbook.org](https://www.extensionbook.org)
